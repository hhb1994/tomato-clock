>一个使用 electron 构建的番茄钟🍅⏰


需要安装 nodejs 以及 npm/yarn

# 依赖安装

```
npm install

//or

yarn

```


# 项目本地运行

```
npm run start

//or

yarn  start
```

# 项目打包

```
npm run make 

//or

yarn  make

```
